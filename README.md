# lightsout

a rendition of [lights out](https://en.wikipedia.org/wiki/Lights_Out_(game)).

## requirements

- norns
- grid

## documention


connect the grid and you'll see a random set of lights.

pressing any lights will it and adjacent lights.

try to turn off all the lights!

also, press the top left button to play :)

## install

install with

```
;install https://github.com/schollz/lightsout
```
